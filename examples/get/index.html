.get(url)
Creates an XHR request to the server with the GET verb.

Code example

                server
                .get("/examples/get/data.txt")
                .success(function (response) {
                    document
                        .getElementById("success")
                        .innerHTML = response;
                })
                .send();

                server
                .get("/foo/bar")
                .fail(function (response) {
                    document
                        .getElementById("fail")
                        .innerHTML = "Failed";
                })
                .send();
