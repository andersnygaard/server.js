server.js

Keep it simple!

Server.js is just a small javascript that handles data to and from the server. 
                
                server.get("/foo/bar/")
                    .every(30).seconds()
                    .success(handleSuccess)
                    .fail(handleFailure)
                    .send();

What can it do?

  • Send XMLHttpRequests to the server with GET, POST, HEAD verbs
  • Cache requests on the client within a session
  • Detect offline and serve cached data when offline
  • Periodical polling
  • Fake endpoints for testing
